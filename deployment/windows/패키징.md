### electron packaging

## 필요한 패키지
*아래는 devDependencies로 설치되어야함
1. electron
2. electron-builder
~~~
npm install electron --save-dev 
npm install electron-builder --save-dev
~~~

## Mac에서 Windows용 빌드하기
1. HomeBrew 설치하기.
2. HomeBrew를 이용해 아래의 패키지 설치.
~~~
brew install wine --without-x11
brew install mono
~~~
